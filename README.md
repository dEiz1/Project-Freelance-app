# Программа: учет заявок фриланс-биржи

## Что внутри
Это готовая учебная программа на Python + SQLite + Tkinter.

Функции:
- автоматическое создание базы данных SQLite;
- 10 связанных таблиц;
- первичные и внешние ключи;
- тестовые данные;
- графический интерфейс;
- форма добавления заявок;
- форма подачи откликов;
- создание договоров;
- учет платежей;
- отчеты;
- готовые SQL-запросы;
- поиск, фильтрация и сортировка;
- резервное копирование базы данных;
- роли пользователей.

## Как запустить
1. Установите Python 3.10+.
2. Скачайте файл `freelance_exchange_app.py`.
3. Откройте папку с файлом.
4. Запустите команду:

```bash
python freelance_exchange_app.py
```

Или дважды нажмите на файл, если Python связан с `.py`.
